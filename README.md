# Capstone Mutual Fund Analytics

## Overview
This project performs Exploratory Data Analysis (EDA) on Mutual Fund data using Python, Pandas, SQLite, and Matplotlib.

## Features
- Data Loading from SQLite
- Data Cleaning
- Risk Category Analysis
- Expense Ratio Analysis
- Fund House Comparison
- Data Visualization

## Technologies
- Python
- Pandas
- SQLite
- Matplotlib
- Jupyter Notebook

## Files
- etl_pipeline.py
- schema.sql
- mutual_fund.db
- notebooks/eda_analysis.ipynb
- requirements.txt

## Conclusion
The project analyzes mutual fund schemes and provides insights into risk categories, expense ratios, and fund performance using visualizations.
